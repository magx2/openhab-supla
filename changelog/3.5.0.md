## 3.5.0

Released version 3.5.0 - THW-01, CCT dimmer commands and channel validity support

### Added
- Added native server support for Zamel THW-01 temperature and humidity sensors.
- Added support for Dimmer CCT command handling.
- Added channel validity-time support, including delayed state cleanup.
- Added `offline`, `valueValidityTimeSec` and `defaultIcon` metadata to native server device channels.
- Added humidity channel tags and weather sensor channel translations.
- Added XML formatting through Spotless.
- Added CI test report uploads.

### Changed
- Updated jSupla server dependency to 4.5.0.
- Renamed common and native server device handler classes for clearer handler naming.
- Passed the whole native `DeviceChannel` when creating openHAB channels and calculating their states.
- Migrated timeout values to `Duration`.
- Mapped jSupla percent values to openHAB `PercentType`.
- Added Celsius units to Heatpol thermostat channel updates.
- Enabled parallel test execution.

### Fixed
- Fixed sub-device parsing so negative raw `subDeviceId` values become `null`.
- Fixed percentage parsing for humidity and other percent-based states.
- Fixed missing parameterized logging argument.
