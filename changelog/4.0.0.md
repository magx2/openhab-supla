## 4.0.0

Released version 4.0.0 - MEW-01 support and jSupla 5 server protocol updates

### Added
- Added native server support for Zamel MEW-01 electricity meters.
- Added MEW-01 integration coverage.
- Added electricity meter phase angle and phase sequence channel handling.
- Added quantity-aware meter states for energy, power, voltage, current, frequency, angle, period, currency and energy price values.

### Changed
- Updated jSupla server dependency to 5.0.0.
- Deciphered native server protocol data in packet order for integration testing.
- Used the meter-provided currency for electricity cost and price-per-unit values.

### Fixed
- Fixed action trigger channel binding for issue #45.
- Fixed duplicate electricity meter phase state mappings for power factor, phase angle and frequency.
