## 3.2.0

Released version 3.2.0 - Handler architecture cleanup and localization

### Added
- Added localization utility support.
- Added Czech, German, Spanish, French, Italian, Polish, Portuguese, Russian and Ukrainian translation bundles.
- Added PlantUML workflow and moved architecture diagrams into the `puml` directory.
- Added AI agent project guidance.

### Changed
- Renamed common cloud and server handler classes to the `SuplaBridge` and `SuplaDevice` abstractions.
- Split server handler traits and responsibilities into clearer packages.
- Removed the `javatuples` dependency and replaced tuple usage with records.
- Changed native server registration flow to throw exceptions instead of returning boolean status flags.
- Set `currentThing` only after successful native server registration.
- Switched native server protocol handling to use `SuplaDeviceChannelValue`.

### Fixed
- Localized channel change failure messages.
- Reported errors when native server auth data cannot be built.
