## 4.0.1

Released version 4.0.1 - Electricity meter compatibility and typed meter channel fixes

### Added
- Added typed channel definitions for meter angle, currency, energy price and time values.
- Added support for simple electricity meter extended values.

### Changed
- Updated jSupla server dependency to 5.0.1.
- Improved electricity meter channel labels, descriptions and openHAB item types.
- Normalized repository line-ending handling.

### Fixed
- Fixed electricity meter extended value dispatch by decoding values according to the packet type.
- Restored compatibility state mapping for electricity meter channels.
- Ignored unknown extended value packet types instead of failing channel updates.
- Stabilized registration status assertions and updated MEW-01 integration expectations.
