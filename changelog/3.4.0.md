## 3.4.0

Released version 3.4.0 - Native server device coverage, optional SSL and integration tests

### Added
- Added native server support and integration coverage for Zamel DIW-01 dimmers.
- Added native server support and integration coverage for Zamel mSLW-01 RGBW controllers.
- Added integration test infrastructure for native server devices, including random port and credential helpers.
- Added openHAB 5.0.0 and 5.1.0 Java TLS helper scripts.
- Added `ChannelType` and channel flag metadata to native server device channels.

### Changed
- Updated jSupla server dependency to 4.3.2.
- Updated the formatter dependency.
- Updated native server handlers to refresh `lastMessageFromDevice` on each device message.
- Refactored handler creation into `SuplaExtension` for integration tests.
- Improved native server logging and state cache usage.
- Disabled Maven transfer progress in CI.

### Fixed
- Allowed the native server bridge to run without SSL.
- Fixed ROW-01 ping handling.
- Fixed and hardened `update-java-tls.sh`.
- Ensured integration tests pick available random ports.
