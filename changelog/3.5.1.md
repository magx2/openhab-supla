## 3.5.1

Released version 3.5.1 - Native server timeout configuration fixes

### Added
- Added timeout configuration parsing tests.
- Added parameterized tests for timeout parsing edge cases.

### Fixed
- Fixed parsing timeout configuration values when openHAB supplies numeric values as doubles.
- Rejected `timeout`, `min` and `max` values less than or equal to zero.
- Reduced noisy ping logging in integration tests.
